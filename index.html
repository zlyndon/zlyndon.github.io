hexo + github page搭建个人博客(windows环境)

闲来无事决定搭建一个自己的个人博客，方便记录、分享自己的学习体会。

从网上查找了各种搭建方式最终选择了现在比较流行的hexo+github来搭建，网上教程都非常罗嗦，讲解一大堆原理，容易混乱。本文最大的简化后旨在快速清晰的傻瓜式搭建，对详细原理感兴趣的同学可以参考其他教程。

搭建思路分为三步：

1.安装、配置hexo

2.注册配置github

3.将hexo同步至github

必须准备工作：

安装Node

由于hexo需要Node.js环境支持
到Node.js官网下载相应最新版本，一路下一步安装即可。

安装git

到git官网下载相应最新版本，一路下一步安装即可。

外网可能下载安装时间较长，也可以搜索下载网上分享的安装包
